Case Model Classification - Part 2 (R)¶

Modelagem¶

Objetivos do projeto: desenvolver um modelo que estime a propensão de um cliente aceitar as ofertas dos produtos nos próximos 30 dias¶

Metodologia:¶

  1. Primeiramente, transformou-se variveis categricas em fatores no R. Como no se sabe o que significa de fato boa parte das variveis, optou-se por assumir que tais variveis binrias (com respostas 1 e 0) so categricas incluindo varivel answer (resposta), carregando-as no R como fatores. Alm disso, a coluna id_category que possui valores inteiros de 0 a 5, tambm foi transformada em fator, j que entende-se que cada nmero representa uma categoria.
  1. Aps esse passo, retirou-se coluna id da base, pois esta serve apenas como identificador do cliente e no remete a nenhuma caracterstica til do cliente para a predio no modelo.
  1. A seguir, uma semente foi definida e separou-se a base em dois conjuntos: treino (80% da base com 738194 registros) e teste (20% da base com 184548), para que o desempenho do modelo consiga ser testado atraves da validao cruzada.
  1. Com bases treino e teste definidas, inicia-se o processo de modelagem utilizando variaes de tipos de modelo, com o intuito de encontrar algum mtodo que melhor defina o evento e seja capaz de prev-lo de forma mais assertiva.
  1. A partir de um erro baixo, comea-se a alterar parmetros de tuning para se tentar obter ganho de predio no mtodo do modelo escolhido.

Obs: como este um problema de classificao, importante citar que para alguns modelos necessita-se escoher uma taxa de corte em relao a probabilidade de chance da resposta ser 1 ou 0. Para isso, uma funo que testa diversos cortes e retorna o que possui maior taxa de acerto foi criada e utilizada.

# Funo para obter melhor valor de corte e respectivo erro
escolhe_corte <- function(probs, real, plotar = FALSE) {
  corte <- seq(0, 1, 0.02)
  erro <- numeric()
  for(i in 1:length(corte)){
    predicao <- as.integer(probs > corte[i])
    
    if(is.numeric(real)){
      erro[i] <- mean((predicao - real)^2)
    }
    else if(is.factor(real)){
      erro[i] <- mean((predicao != real))
    }
    
  }
  if(plotar){
    plot(corte, erro, type = "l")
    abline(v=corte[which.min(erro)], col="red")
  }
  
  return(c(corte = corte[which.min(erro)], erro = min(erro)))
}

predict.regsubsets <- function(object, newdata, id,...){
  form <- as.formula(object$call[[2]])
  matriz <- model.matrix(form, newdata)
  coeficientes <- coef(object, id = id)
  
  return(matriz[, names(coeficientes)] %*% coeficientes)
  
}
